  • Is There A White Sox Game Today
  • www.isthereawhitesoxgametoday.com
    • This is probably both the simplest and most popular website I have made. Based on isthereacubsgametoday.com, this is a website that helps people trying to get around the south side of Chicago know whether or not they'll have to deal with the traffic backups and service disruptions that come along with the crowd at U.S. Cellular Field. What I really like about this web app is that it demonstrates how a small amount of code can be used to make something extremely useful.

    • Technical Summary

      Built in Ruby on top of Sinatra. View it on Github.
  • exquisite chat
  • fierce-fog-4004.herokuapp.com
    • This project is an attempt at creating a digital version of an exquisite corpse, an old surrealist technique for creating collaborative art. Users invite their friends into a room where they all contribute lines to a poem, with the caveat that they only ever see the last line the person before them wrote, until the entire poem is revealed at the end of the session. There's a lot of interesting projects happening in the digital poetics world. exquisite chat is one way I'm trying to explore how the web can be used as a medium for creative expression and collaboration.

    • Technical Summary

      Built using node.js and making use of Redis data store.
  • Chicago Crime Network Visualization
  • nameless-coast-5335.herokuapp.com
    • This network visualization displays crimes that occurred in Chicago on June 24, 2012, one of the days with the most violent crimes of the year. The City of Chicago Data Portal makes a lot of data available. My hope is that visualizations like this encourage users to interact with and explore the data in ways that are not possible through traditional interfaces. I think this also represents what I consider an exciting new place in web standards, where developers are able to create interesting and novel interfaces and data visualizations that are available to anyone with a web browser.

    • Technical Summary

      This is a purely client side project making use of d3, based on Jim Vallandingham's tutorial on making network visualizations. A Ruby script was used to transform the JSON from the Chicago data portal into a form more easily consumed by d3.
